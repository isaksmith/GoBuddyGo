UI Design Prompt — Mobile Racing Game
Overall Visual Language:
Dark navy/teal background (#0d2a3d to #1a3a52 range). The aesthetic is vibrant arcade — think neon glow meets cartoon polish. Everything has rounded corners, bold outlines, and feels chunky and tactile.

Screen 1 — Splash/Login Screen:

Full bleed diagonal racing scene as background art (top-down perspective race cars with motion blur/rocket trails)
No header, just artwork filling the screen
Two CTA buttons at the bottom, centered, stacked vertically:

"CONNECT TO FACEBOOK" — white outlined rounded rectangle, blue text, icon left
"PLAY OFFLINE" — coral/orange-red filled rounded rectangle, white bold uppercase text


Button style: pill-shaped, bold uppercase sans-serif, slight drop shadow

Screen 2 — Main Game/Lobby Screen:

Top HUD bar:

Left: circular icon button with notification badge (red dot, number)
Center: "PETROL" label + horizontal progress bar (orange fill, dark track) + "20 L" text
Right: circular icon button (trash/delete style)
Below that row: coin count ("120") with gold coin icon
Far right of second row: circular compass/crosshair icon button


Center: The actual game lane — 3 vertical lanes with gradient color washes (green, yellow/rainbow, orange). Each lane glows. Cars are top-down cartoon style, centered in lanes with neon underglow effects
Bottom action area:

"INVITATIONS" button — teal/green rounded pill, with red badge showing count ("5")
"PLAY" button — coral/red-orange large rounded pill, white bold uppercase text — PRIMARY CTA
Bottom row: settings gear icon (left) | "RANDOM PLAY" outlined pill button (center) | bar chart icon (right)



Screen 3 — Upgrades/Shop Screen:

Back chevron < top left, "UPGRADES" title centered (white, bold), icon button top right
2-column grid of upgrade cards:

Each card: large circular icon (gradient blue background, cartoon icon inside), red notification badge on some
Label below circle (e.g., "Longer", "Slow Down", "Rockets", "Coins")
Price below label (gold coin icon + number, e.g., "30")
Action button below: either orange "USE" pill or teal "BUY" pill — bold white uppercase text


Grid has consistent spacing, cards feel like tiles


Global Design Tokens:

Background: #0e2a40 (deep navy)
Primary action: #f4633a (coral-orange)
Secondary action: #3ecf8e (teal-green)
Accent: #f5c518 (gold/yellow)
Text: white, bold, uppercase for labels/buttons
Cards/panels: slightly lighter navy with subtle blue tint, rounded corners (~16-20px radius)
Icons: enclosed in circular containers with gradient fills
Notification badges: red circle, white number, top-right of parent element
Font feel: rounded, bold, game-UI — something like Nunito, Fredoka One, or Baloo
All buttons: pill-shaped (border-radius ~50px), uppercase, bold, no lowercase anywhere


Color palette (bottom of image): coral red, amber orange, yellow, teal, periwinkle blue, lavender purple, slate blue
